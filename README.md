# NOMMADE

Este es mi repositorio de **Nommade**, una plataforma de e-commerce diseñada para el comercio de bebidas, enfocada en una estética un poco moderna y una experiencia de usuario fluida.

Este proyecto integra un desarrollo **Full Stack** con arquitectura cliente-servidor, base de datos y autenticación.

# Tecnologías Utilizadas:

# Frontend:

- **HTML5 & CSS3:** Estructura semántica y diseño moderno.
- **SASS:** Arquitectura de estilos modular con variables y mixins.
- **Bootstrap 5:** Framework para una interfaz responsiva.
- **JavaScript:** Lógica del carrito de compras, manipulación del DOM y fetch.

# Backend:

- **Node.js & Express:** Servidor y manejo de rutas.
- **MySQL:** Base de datos relacional para gestión de productos y usuarios.
- **JWT (JSON Web Tokens):** Manejo de sesiones.
- **BcryptJS:** Hashing de contraseñas.

---

# Características:

- **Sistema de Autenticación:** Registro y Login con validación en el servidor y gestión de cookies.
- **Carrito de Compras:** Cálculo de totales y gestión de cantidades.
- **Base de Datos:** Consultas optimizadas para productos, promociones y gestión de stock.
- **Diseño Responsivo:** Adaptado para dispositivos móviles, tablets y escritorio.

# Limitaciones:

1.  **Seguridad incompleta:** Falta implementar una capa de validación más estricta. Al no tener una seguridad 100% asegurada, la página está expuesta a ataques, y por este motivo no es buen momento para un despliegue de ella.
2.  **Despliegue:** Debido a que el proyecto cuenta con un backend dinámico, GitHub Pages no es compatible. El siguiente paso es el despliegue.
3.  **Pasarela de Pago:** La integración con APIs reales es la funcionalidad pendiente.
